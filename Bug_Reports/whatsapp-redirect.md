# Bug Report: WhatsApp Redirect Page

Bug ID: BUG-02

Module:
Order on WhatsApp button

Description:
Clicking the "Order on WhatsApp" button redirects the user to the WhatsApp web landing page instead of directly opening a chat window.

Steps to Reproduce:
1. Open the website
2. Click "Order on WhatsApp"
3. Observe the redirect page

Expected Result:
The button should directly open WhatsApp chat with the business number.

Actual Result:
User is first redirected to WhatsApp web landing page.

Severity:
Low

Priority:
Medium

Environment:
Desktop browser
